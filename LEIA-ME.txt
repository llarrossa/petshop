🗂️ ESTRUTURA DE PASTAS
═══════════════════════════════════════════════════════════════

📁 /config/

   └─ schema.sql              💾 Script de criação do banco

📁 /classes/
   ├─ Tutor.class.php         👥 Gestão de tutores/clientes
   ├─ Pet.class.php           🐕 Gestão de pets
   ├─ Produto.class.php       📦 Gestão de produtos
   ├─ Servico.class.php       ✂️ Gestão de serviços
   ├─ Profissional.class.php  👨‍⚕️ Gestão de profissionais
   ├─ Agenda.class.php        📅 Sistema de agendamentos
   └─ Venda.class.php         💰 PDV e vendas

📁 /controllers/
   ├─ tutores.php             📋 Controller de tutores
   └─ produtos.php            📋 Controller de produtos
   └─ (adicione mais conforme necessário)

📁 /views/
   ├─ template.php            🎨 Template base do sistema
   └─ /tutores/
       ├─ list.php            📋 Listagem
       ├─ form.php            📝 Formulário
       └─ view.php            👁️ Visualização

📁 /public/
   ├─ index.php               🏠 Página inicial
   ├─ login.php               🔐 Tela de login
   ├─ authenticate.php        🔑 Autenticação
   ├─ logout.php              🚪 Logout
   ├─ dashboard.php           📊 Dashboard principal
   ├─ /css/
   │   └─ style.css           🎨 Estilos CSS
   ├─ /js/
   │   └─ script.js           ⚡ JavaScript
   └─ /uploads/               📤 Arquivos enviados

📁 /helpers/
   └─ functions.php           🛠️ Funções auxiliares

📄 .htaccess                  🔒 Configurações Apache
📄 index.php                  🏁 Ponto de entrada
